# Welcome to StoryFilm!

여러분은 언제 쓰나요? 
**스토리필름**은 일기를 **꾸준히** 쓸 만큼 재미있고 자신을 돌아보게 하는 **일기 작성 서비스**입니다.


## 메인기능 - 일기 작성
<img width="1102" alt="image" src="https://user-images.githubusercontent.com/87116017/193998778-f7e30e85-5d99-4303-81c8-e1e14b0f99db.png">
사진을 넣고 일기를 작성합니다.
작성한 일기는 필름으로 저장되며 열람이 불가합니다.

## 메인기능 - 일기 인화
<img width="1110" alt="image" src="https://user-images.githubusercontent.com/87116017/193999123-c8e54281-372b-41bc-86d7-4ff51dd26b0f.png">
일기를 작성해 필름을 모두 채우면 인화가 진행됩니다. 
가려졌던 일기 확인이 가능하고, 작성한 일기를 돌아보며 코멘트를 남길 수 있습니다.

## 메인기능 - 보관함
<img width="1061" alt="image" src="https://user-images.githubusercontent.com/87116017/193999381-935c4c28-3969-4e09-a8fe-5e4e72b624a2.png">
인화가 완료된 일기의 필름통은 필름 보관함에 저장되며, 인화된 일기는 사용자의 기준으로 분류된 앨범 보관함에서 쉽게 찾아 읽을 수 있습니다.
